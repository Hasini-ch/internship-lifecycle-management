# Internship Lifecycle Management

## Project Overview
Internship Lifecycle Management is a ServiceNow application that automates the complete internship process, replacing manual emails and spreadsheets with a centralized workflow.

## Problem Statement
Managing internships manually leads to delays, poor communication, duplicate records, and difficulty tracking approvals and student progress. This application digitizes the entire internship lifecycle.

## Features
- Student internship application submission
- Faculty approval workflow
- Company approval workflow
- Mentor assignment
- Internship status tracking
- Certificate management
- Role-based access control
- Service Catalog request
- Automated workflows using Flow Designer
- Reports

## Workflow
1. Student submits internship application.
2. Faculty reviews and approves.
3. Company reviews and approves.
4. Mentor is assigned.
5. Internship status changes to Active.
6. Certificate is generated after completion.
